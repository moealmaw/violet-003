# Project Context: Violet M. Digby — Artist Website

## Overview
This project is a commemorative website for **Violet M. Digby (1900–1960)**, a British artist known for her work in India, Kashmir, and St Ives. The site serves as a biographical archive and a gallery, designed to feel like an intimate exhibition catalogue.

## Core Objectives
- Create a visually stunning, "painterly" experience that reflects Violet's artistic style.
- Provide a comprehensive biographical narrative across different life chapters.
- Showcase her paintings in a thematic gallery.
- Map her artistic journey through her geographies.
- Maintain an archival record of her exhibitions and legacy.

## Tech Stack & Guidelines
- **Framework:** Next.js (as per project bootstrap)
- **Routing:** React Router (as per `prompt.md` requirements)
- **Styling:** Tailwind CSS (exclusive use)
- **Components:** shadcn/ui, Lucide React icons
- **Architecture:** Strict component-based architecture, atomic design for UI primitives.
- **Data Management:** Centralized `data.ts` for all static and mock data.
- **Theming:** Support for light and dark mode with a toggle.
- **Responsiveness:** Fully functional on all screen sizes.
